# Master context addendum - Academia sync gate

Fecha: 2026-07-04

## Proposito

Este addendum complementa el documento maestro de continuidad CXOrbia TyA.

## Regla nueva

Al revisar, corregir, ampliar o conectar cualquier modulo, tambien se debe revisar si Academia necesita actualizar:

- manuales;
- cursos;
- rutas de aprendizaje por rol;
- glosario;
- guias paso a paso;
- errores frecuentes;
- checklist de validacion;
- temas para solicitud de capacitacion;
- contenido comercial y de venta cruzada si aplica.

## Instruccion para cada bloque futuro

Antes de cerrar un bloque, verificar:

1. Que modulo cambio.
2. Que roles afecta.
3. Que manual o curso debe cambiar.
4. Si ya se documento la actualizacion o el pendiente.
5. Si Claude debe revisar algo puntual cuando vuelva a tener capacidad.

## Instruccion para nuevos candidatos de Claude

Cada auditoria de prototipo debe incluir un apartado Academia:

- que cambios afectan manuales;
- que cambios afectan cursos;
- que rutas por rol deben actualizarse;
- que falta documentar;
- que mejora comercial o de onboarding se detecto.
