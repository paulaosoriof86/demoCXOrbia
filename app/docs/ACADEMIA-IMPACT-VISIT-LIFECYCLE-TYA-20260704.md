# Academia impact - Visit lifecycle TyA

Fecha: 2026-07-04

## Objetivo

Aplicar el gate de Academia al bloque de visitas, agenda, reprogramacion, cancelacion, visita realizada y cuestionario realizado.

## Manuales a actualizar o crear

1. Manual shopper de visitas disponibles y agenda.
2. Manual shopper de reprogramacion y cancelacion.
3. Manual shopper de visita realizada y cuestionario realizado.
4. Manual operativo de seguimiento de visitas.
5. Manual admin de lifecycle y auditoria de estados.
6. Manual cliente de lectura de estados.

## Cursos relacionados

### Shopper

Curso: Ejecutar una visita correctamente.

Lecciones:

1. Como ver visitas disponibles.
2. Como leer instructivo y reglas.
3. Como agendar una visita.
4. Como reprogramar si aplica.
5. Como cancelar si aplica.
6. Como marcar visita realizada.
7. Como completar o abrir cuestionario.
8. Que significa cuestionario realizado.
9. Diferencia entre cuestionario realizado, submitido, liquidacion y pago.
10. Errores frecuentes.

### Operativo / coordinador

Curso: Seguimiento diario de visitas.

Lecciones:

1. Como monitorear visitas agendadas.
2. Como revisar reprogramaciones.
3. Como validar cancelaciones.
4. Como detectar visita realizada sin cuestionario.
5. Como escalar conflictos.
6. Como usar notificaciones.

### Admin

Curso: Control del ciclo de vida de visitas.

Lecciones:

1. Estados canonicos de visita.
2. Diferencia entre estado operativo y estado financiero.
3. Como auditar fechas.
4. Como evitar liquidaciones prematuras.
5. Como revisar conflictos HR/plataforma.

### Cliente final

Curso: Lectura de avance de visitas.

Lecciones:

1. Que significa visita disponible.
2. Que significa visita agendada.
3. Que significa visita realizada.
4. Que significa en revision.
5. Como leer avance sin acceso a datos sensibles.

### Consultora / representante / franquiciado / aliado / socio

Curso: Valor de la trazabilidad de visitas.

Lecciones:

1. Como el lifecycle reduce reprocesos.
2. Como mejora el control operativo.
3. Como ayuda a explicar avance al cliente.
4. Como habilita paquetes de automatizacion y reportes.

## Glosario requerido

- Visita disponible.
- Visita asignada.
- Agenda.
- Reprogramacion.
- Cancelacion.
- Visita realizada.
- Cuestionario realizado.
- Submitido.
- Revision admin.
- Liquidacion.
- Pago.
- Conflicto HR/plataforma.

## Checklist interactivo recomendado

Para shopper:

- Revise instructivo.
- Confirme reglas de fecha/franja.
- Agende.
- Realice visita.
- Suba/registre evidencias si aplica.
- Complete cuestionario.
- Verifique estado pendiente de revision.

Para ops/admin:

- Revise agenda.
- Valide fechas.
- Revise reprogramaciones/cancelaciones.
- Detecte visitas realizadas sin cuestionario.
- Escale conflicto si hay discrepancia.
- No habilite liquidacion sin revision/submitido.
