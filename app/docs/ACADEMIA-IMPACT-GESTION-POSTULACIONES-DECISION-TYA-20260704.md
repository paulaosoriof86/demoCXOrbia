# Academia impact - Gestion de postulaciones y decision TyA

Fecha: 2026-07-04

## Objetivo

Aplicar el gate de Academia al flujo operativo de gestion de postulaciones.

## Manuales a actualizar o crear

1. Manual operativo de gestion de postulaciones.
2. Manual admin de aprobacion estrategica.
3. Manual shopper de resultados de postulacion.
4. Manual de notificaciones WhatsApp Web / in-app.
5. Manual de ajuste y confirmacion de fecha.

## Cursos relacionados

### Operativo / coordinador

Curso: Gestion estrategica de postulaciones.

Lecciones:

1. Como agrupar postulaciones por sucursal.
2. Como agrupar por shopper.
3. Como revisar postulaciones multiples del mismo shopper.
4. Como decidir asignacion por Q1/Q2 o periodo.
5. Como pedir ajuste de fecha.
6. Como pedir confirmacion de fecha.
7. Como pedir reprogramacion.
8. Como enviar mensaje suave cuando no se selecciona.
9. Como evitar doble aprobacion.

### Admin

Curso: Supervision de postulaciones y asignaciones.

Lecciones:

1. Como auditar quien aprobo.
2. Como revisar sucursales con multiples postulados.
3. Como balancear shoppers entre visitas.
4. Como validar restricciones antes de aprobar.
5. Como revisar notificaciones.

### Shopper

Curso: Que pasa despues de postularme.

Lecciones:

1. Que significa postulacion pendiente.
2. Que significa que me pidan ajustar fecha.
3. Que significa que me pidan confirmar fecha.
4. Que significa no ser seleccionado en esta ocasion.
5. Que hacer cuando me aprueban.
6. Como programar/reprogramar/cancelar.

## Glosario requerido

- Postulacion pendiente.
- Fecha propuesta.
- Ajuste de fecha.
- Confirmacion de fecha.
- No seleccionado en esta ocasion.
- Aprobado con fecha.
- Aprobado pendiente de programar.
- Doble aprobacion.
- WhatsApp Web.
- Mensajeria futura.

## Checklist interactivo recomendado

Para ops/admin:

- Agrupe por sucursal.
- Revise postulados por shopper.
- Revise restricciones.
- Compare Q1/Q2 o periodo.
- Decida aprobar, pedir ajuste, pedir confirmacion o no seleccionar.
- Registre motivo.
- Verifique que la visita sale de disponibles.
- Verifique notificacion preparada/enviada.

Para shopper:

- Revise notificacion.
- Si piden ajuste, proponga fecha nueva.
- Si piden confirmacion, confirme asistencia.
- Si fue aprobado, programe o revise fecha aprobada.
