# Academia impact - Email traceability TyA

Fecha: 2026-07-04

## Objetivo

Aplicar el gate de Academia al bloque de correo y trazabilidad.

## Manuales a actualizar o crear

1. Manual admin de configuracion de correo.
2. Manual operativo de log manual de correos.
3. Manual de asociacion de correos a gestiones.
4. Manual de privacidad y datos sensibles en correo.
5. Manual de diferencia entre notificaciones y correo.

## Cursos relacionados

### Admin / superadmin

Curso: Configuracion y trazabilidad de correo.

Lecciones:

1. Por que registrar trazabilidad por correo.
2. Que es una cuenta de correo del tenant.
3. Que es log manual.
4. Que es integracion futura Gmail/Outlook/IMAP.
5. Que datos se pueden guardar.
6. Que datos requieren politica de privacidad.
7. Como asociar correo a visita, cliente, shopper, soporte o pago.

### Operativo / coordinador

Curso: Seguimiento por correo.

Lecciones:

1. Como registrar un correo manualmente.
2. Como vincularlo a una gestion.
3. Como clasificarlo.
4. Como detectar si requiere privacidad/revision.
5. Como usarlo como soporte de seguimiento.

### Consultora / aliado / socio

Curso: Valor de la trazabilidad por correo.

Lecciones:

1. Como mejora auditoria.
2. Como reduce perdida de informacion.
3. Como complementa WhatsApp y notificaciones.
4. Como ayuda en soporte, clientes y pagos.

## Glosario requerido

- Email traceability.
- Log manual.
- Thread.
- Message.
- Provider.
- OAuth futuro.
- IMAP/SMTP futuro.
- Metadata only.
- Snippet only.
- Adjuntos.
- Revision de privacidad.

## Checklist interactivo recomendado

Para admin/ops:

- Seleccione gestion relacionada.
- Registre asunto/remitente/destinatario.
- Clasifique tipo de correo.
- Determine si contiene datos sensibles.
- Defina si se guarda metadata o snippet.
- Vincule a entidad.
- Guarde trazabilidad.
