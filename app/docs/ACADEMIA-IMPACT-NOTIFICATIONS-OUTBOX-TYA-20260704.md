# Academia impact - Notifications outbox TyA

Fecha: 2026-07-04

## Objetivo

Aplicar el gate de Academia al bloque de notificaciones, WhatsApp Web fallback, plantillas y outbox.

## Manuales a actualizar o crear

1. Manual operativo de notificaciones.
2. Manual de WhatsApp Web fallback.
3. Manual de plantillas por evento.
4. Manual admin de auditoria de mensajes.
5. Manual shopper de notificaciones recibidas.
6. Manual Academia de notificaciones de cursos/manuales.

## Cursos relacionados

### Operativo / coordinador

Curso: Notificaciones operativas y WhatsApp Web.

Lecciones:

1. Que es una notificacion in-app.
2. Que es WhatsApp Web fallback.
3. Como preparar un mensaje.
4. Como confirmar envio manual.
5. Como pedir ajuste de fecha.
6. Como pedir confirmacion.
7. Como pedir reprogramacion.
8. Como enviar mensaje suave de no seleccionado.
9. Como revisar contacto faltante.

### Admin

Curso: Auditoria de notificaciones.

Lecciones:

1. Estados de entrega.
2. Diferencia entre preparado y enviado.
3. Diferencia entre manual y API futura.
4. Como revisar plantillas.
5. Como auditar actor, canal y resultado.

### Shopper

Curso: Como leer mis notificaciones.

Lecciones:

1. Aprobacion de postulacion.
2. Ajuste de fecha solicitado.
3. Confirmacion de fecha.
4. Reprogramacion.
5. Cuestionario pendiente.
6. Curso o certificacion asignada.

### Cliente final

Curso: Notificaciones del portal cliente.

Lecciones:

1. Que alertas puede recibir.
2. Como interpretar novedades y reportes.
3. Como solicitar soporte o capacitacion.

### Academia

Curso: Notificaciones de aprendizaje.

Lecciones:

1. Curso nuevo asignado.
2. Manual actualizado.
3. Leccion nueva.
4. Solicitud de capacitacion.
5. Contenido pendiente de revision.

## Glosario requerido

- Notificacion in-app.
- Outbox.
- WhatsApp Web fallback.
- Make.
- API de mensajeria.
- Envio manual confirmado.
- Contacto faltante.
- Plantilla de mensaje.
- Gate apagado.

## Checklist interactivo recomendado

Para ops/admin:

- Seleccione evento.
- Revise destinatario.
- Revise plantilla.
- Confirme canal.
- Si es WhatsApp Web, copie/abra mensaje.
- Confirme envio manual si lo realizo.
- Revise estado final.
