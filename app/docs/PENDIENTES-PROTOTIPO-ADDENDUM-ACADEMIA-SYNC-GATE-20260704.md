# Pendientes prototipo - Addendum Academia sync gate

Fecha: 2026-07-04

## Pendiente permanente

Cada vez que se modifique o amplie un modulo, el prototipo debe revisar si Academia necesita actualizacion.

## A revisar por cambio

- Manual del modulo.
- Curso relacionado.
- Ruta de aprendizaje por rol.
- Guia paso a paso.
- Botones y acciones.
- Errores frecuentes.
- Checklist de validacion.
- Glosario.
- Temas para solicitar capacitacion.
- Valor comercial y paquetes adicionales si aplica.

## Roles

- Superadmin.
- Admin.
- Operativo/coordinador.
- Shopper.
- Cliente.
- Consultora/representante/franquiciado/aliado/socio.

## Regla

Si el modulo cambio y Academia no se actualizo, debe quedar como pendiente explicito.
