# Regla propuesta unificacion de fuentes TyA

Fecha: 2026-07-03

## Objetivo

Evitar doble conteo entre fuentes relacionadas antes de cualquier carga futura.

## Regla base

- Usar HR como fuente principal de visitas y fechas.
- Usar postulaciones como fuente de aplicacion/asignacion.
- Usar fuentes auxiliares solo para trazabilidad.

## Criterio de comparacion

Comparar proyecto, periodo, pais, visita, persona, estado y fecha cercana.

## Tratamiento

- No registrar dos veces el mismo evento.
- Dejar casos dudosos en revision.
- Generar reporte antes de avanzar.

## Resultado esperado

Trazabilidad sin duplicacion operativa.
