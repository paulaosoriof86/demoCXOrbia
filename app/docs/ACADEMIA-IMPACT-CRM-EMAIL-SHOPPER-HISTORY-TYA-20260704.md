# Academia impact - CRM, email and shopper history TyA

Fecha: 2026-07-04

## Objetivo

Aplicar el gate de Academia al bloque de correo real futuro por usuario, CRM con OneDrive/documentos y shopper communication history.

## Manuales a actualizar o crear

1. Manual de correo por usuario.
2. Manual de trazabilidad email/WhatsApp.
3. Manual CRM cliente/prospecto/proyecto.
4. Manual OneDrive externo y documentos.
5. Manual shopper history completo.
6. Manual privacidad y visibilidad de comunicaciones.

## Cursos relacionados

### Admin / superadmin

Curso: Configuracion de comunicacion y trazabilidad.

Lecciones:

1. Correo por usuario.
2. Crear usuario con o sin correo vinculado.
3. Diferencia entre log manual e integracion real.
4. Vincular correo a cliente/proyecto/shopping/support.
5. Privacidad y permisos.
6. OneDrive como carpeta externa.
7. Pipeline y documentos de cliente.

### Operativo / coordinador

Curso: Trazabilidad operativa.

Lecciones:

1. Registrar comunicacion por WhatsApp.
2. Registrar comunicacion por correo.
3. Vincular comunicacion a shopper o visita.
4. Ver historial shopper.
5. Distinguir comunicacion visible al shopper vs interna.

### Shopper

Curso: Mi historial y mis comunicaciones.

Lecciones:

1. Que comunicaciones puedo ver.
2. Que significa notificacion, WhatsApp o correo registrado.
3. Como revisar mis visitas historicas.
4. Como ver certificaciones, liquidaciones y pagos cuando apliquen.

### Consultora / franquicia / aliado

Curso: CRM y trazabilidad comercial.

Lecciones:

1. Diferencia entre consultora, cliente final y proyecto.
2. Como usar pipeline.
3. Como vincular carpetas OneDrive.
4. Como conservar propuestas y hojas de costos.
5. Como retomar prospectos con historial completo.

## Checklist interactivo recomendado

Para admin:

- Crear usuario.
- Definir si tiene correo vinculado.
- Configurar permisos.
- Validar privacidad.
- Vincular comunicacion a entidad.

Para CRM:

- Crear prospecto/cliente.
- Asociar carpeta OneDrive.
- Registrar documentos.
- Registrar pipeline.
- Asociar correos/notificaciones.
- Mantener historial.

Para shopper history:

- Importar visitas historicas.
- Registrar postulaciones/asignaciones.
- Registrar comunicaciones.
- Definir visibilidad.
- Proteger notas internas.
