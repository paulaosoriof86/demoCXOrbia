## Regla para futuras mejoras de prototipo

Cada modulo debe tener contenido de capacitacion por rol:

- que es;
- para que sirve;
- que valor agrega;
- como se opera paso a paso;
- que botones o acciones usa;
- errores comunes;
- validacion esperada;
- cuando pedir capacitacion;
- beneficios y venta cruzada si aplica.

## Roles cubiertos

- Superadmin.

- Operativo/coordinador.
- Shopper.
- Cliente final.
- Consultora, representante, franquiciado, aliado o socio.
