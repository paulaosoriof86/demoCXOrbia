# Academia impact - Restricciones y ficha de postulacion TyA

Fecha: 2026-07-04

## Objetivo

Aplicar el gate de Academia al bloque de restricciones de proyecto y ficha de postulacion.

## Manuales a actualizar o crear

1. Manual shopper de ficha de postulacion.
2. Manual shopper de restricciones y elegibilidad.
3. Manual operativo de validacion de restricciones.
4. Manual admin de configuracion de restricciones por proyecto.
5. Manual cliente de requisitos de visita.
6. Manual consultora/aliado de configuracion comercializable de proyectos.

## Cursos relacionados

### Shopper

Curso: Como leer una ficha de postulacion.

Lecciones:

1. Que informacion debo revisar antes de postularme.
2. Que es honorario.
3. Que es reembolso.
4. Que es escenario.
5. Que es resumen del proyecto.
6. Que son restricciones de perfil.
7. Que es frecuencia o ultima visita.
8. Que es franja, horario o dias permitidos.
9. Que significa requisito del cliente.
10. Que significa medicion interna.
11. Que hacer si no cumplo una restriccion.

### Operativo / coordinador

Curso: Validar restricciones en postulaciones.

Lecciones:

1. Como revisar elegibilidad del shopper.
2. Como validar restriccion de ultima visita.
3. Como validar edad/perfil si el proyecto lo requiere.
4. Como validar horario/dias/franja.
5. Como diferenciar requisito del cliente vs control interno.
6. Como escalar revision manual.

### Admin

Curso: Configurar restricciones por proyecto.

Lecciones:

1. Como crear restricciones de perfil.
2. Como crear restricciones de frecuencia.
3. Como configurar ventanas de dias/horarios.
4. Como marcar una restriccion como requisito del cliente.
5. Como marcar una regla como medicion interna.
6. Como evitar hard-codear reglas de un proyecto en todos.

### Cliente final

Curso: Requisitos de visita y control de calidad.

Lecciones:

1. Como sus requisitos aparecen en la plataforma.
2. Como las restricciones ayudan a ejecutar mejor el proyecto.
3. Como se diferencian requisitos del cliente y controles internos.

### Consultora / representante / franquiciado / aliado / socio

Curso: Proyectos configurables y valor comercial.

Lecciones:

1. Como adaptar reglas por cliente.
2. Como reducir errores de postulacion.
3. Como explicar honorarios, escenario y restricciones.
4. Como esto mejora adopcion y venta cruzada.

## Glosario requerido

- Ficha de postulacion.
- Honorario.
- Reembolso.
- Escenario.
- Restriccion de perfil.
- Elegibilidad.
- Ultima visita.
- Frecuencia.
- Requisito del cliente.
- Medicion interna.
- Revision manual.

## Checklist interactivo recomendado

Para shopper:

- Lei resumen del proyecto.
- Revise honorario y reembolso.
- Entendi el escenario.
- Revise restricciones de perfil.
- Revise fechas, dias y horarios.
- Revise frecuencia o ultima visita.
- Confirme si cumplo requisitos.

Para ops/admin:

- Revise elegibilidad.
- Revise restricciones de fecha y horario.
- Revise frecuencia/ultima visita.
- Identifique si es requisito cliente o control interno.
- Decidi si aplica revision manual.
