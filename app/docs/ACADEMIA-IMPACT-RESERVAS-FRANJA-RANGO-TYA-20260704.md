# Academia impact - Reservas, franja y rango TyA

Fecha: 2026-07-04

## Objetivo

Aplicar el gate de Academia al bloque de reservas, fecha propuesta, agendamiento, reprogramacion, franja, quincena y autorizaciones especiales.

## Manuales a actualizar o crear

1. Manual shopper de reservas y fecha propuesta.
2. Manual shopper de agendamiento por franja.
3. Manual shopper de reprogramacion correcta.
4. Manual operativo de validacion de fechas.
5. Manual admin de autorizaciones fuera de rango/franja.
6. Manual de impacto en puntaje shopper.

## Checklist interactivo recomendado

Para shopper:

- Revise disponible desde.
- Verifique si la visita es WK o WKND.
- Verifique quincena.
- Proponga fecha valida.
- Si necesita fecha fuera de rango, explique motivo.

Para ops/admin:

- Revise availableFrom.
- Valide franja.
- Valide quincena.
- Si falla regla, solicite motivo.
- Registre autorizador.
- Clasifique si hay impacto en puntaje o no.
- Escale si falta criterio de ranking.
